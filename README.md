# SchoolDiary2
Альтернативный клиент для электронного дневника АСУ РСО (для жителей Самарской области)
## Использование
Чтобы использовать приложение Школьный дневник, нужно:
 1. Иметь стабильное (желательно) подключение к сети Интернет
 2. Войти в личный кабинет ученика
  2.1. Найти и выбрать вашу школу в выпадающем списке поля "Школа"
  2.2. Ввести ваше имя пользователя/фразу логина и пароль в соответствующие поля
