# SchoolDiary2
Альтернативный клиент для электронного дневника АСУ РСО (для жителей Самарской области)
## Использование
Чтобы использовать приложение Школьный дневник, нужно:
<ol>
  <li>Иметь стабильное (желательно) подключение к сети Интернет</li>
  <li>Войти в личный кабинет ученика
    <ol>
      <li>Найти и выбрать вашу школу в выпадающем списке поля "Школа"</li>
      <li>Ввести ваше имя пользователя/фразу логина и пароль в соответствующие поля</li>
    </ol>
  </li>
</ol>
